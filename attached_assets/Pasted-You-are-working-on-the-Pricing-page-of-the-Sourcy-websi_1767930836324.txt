# GOAL

When a user clicks **“Get Started”** on **Voice Agents**, open a modal that allows them to:

1) View the available **Voice Agents**
2) Select one or multiple Voice Agents
3) Optionally increase quantity per agent
4) See pricing split into:

   - Monthly partnership fee
5) See totals update in real time
6) Proceed to **Stripe Checkout** by clicking “Buy”
