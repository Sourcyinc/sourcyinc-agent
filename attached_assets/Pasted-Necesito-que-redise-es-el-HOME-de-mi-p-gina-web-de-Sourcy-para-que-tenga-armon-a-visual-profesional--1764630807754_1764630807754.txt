Usa esta paleta en TODA la página:

- Fondo secundario para cards: #1e293b (slate-800)
- Bordes sutiles: rgba(51, 65, 85, 0.5) o #334155
- Texto principal: #ffffff
- Texto secundario: #94a3b8 (slate-400)
- Acento primario: #06b6d4 (cyan-500)
- Acento secundario: #2dd4bf (teal-400)

## ESTILO DE CARDS UNIFICADO
Todas las cards deben seguir este estilo:
```css
.card {
  background-color: #1e293b;
  border: 1px solid rgba(51, 65, 85, 0.5);
  border-radius: 16px;
  padding: 24px;
  transition: all 0.3s ease;
}

.card:hover {
  border-color: rgba(6, 182, 212, 0.5);
  box-shadow: 0 0 30px rgba(6, 182, 212, 0.1);
}
```
